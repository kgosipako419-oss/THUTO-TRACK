# ThutoTrack

**Parent-Teacher-Student Progress Tracker for Botswana Schools**

## About

ThutoTrack is a planned system to help schools in Botswana digitize student marks and behavior records, and let parents track progress in real-time.

The goal is to make it easy for teachers to record results and for parents to see how their children are doing during the term, without waiting for report cards.

## Why This Matters

- Most schools still use paper or Excel for marks
- Parents only see results once per term
- Rural parents have no way to track performance or behavior
- Botswana’s SmartBots and BETP programs are pushing for digital education solutions

## Planned Features

**For Teachers**
- Enter marks and attendance offline, sync when online
- Add quick behavior notes
- Bulk upload via Excel
- Generate term report PDFs

**For Parents**
- Check marks and attendance via USSD codes
- Get updates via WhatsApp and SMS
- View progress over time on web/mobile

**For Schools**
- Manage classes, students, teachers
- View school-wide performance
- Export data for Ministry submissions

## Tech Ideas

- Backend: Django + PostgreSQL
- Frontend: React for web, React Native for Android
- Messaging: USSD and WhatsApp via Africa’s Talking API
- Offline-first design for low connectivity areas
